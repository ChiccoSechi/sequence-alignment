Needleman-Wunsch Algorithm

The Needleman-Wunsch algorithm finds the optimal alignment between two sequences across their entire length: it is a global alignment algorithm.

It operates in three steps:

  1. Initialization: a matrix of size (sequence1_length + 1) × (sequence2_length + 1) is created. The first row and column are filled with cumulative gap penalties.
  2. Fill the matrix: 
    Score[i,j] = max(
    score[i-1, j]   + gap_penalty,   # top      (gap in sequence 2)
    score[i-1, j-1] + s(aᵢ, bⱼ),     # diagonal (match/mismatch)
    score[i, j-1]   + gap_penalty    # left     (gap in sequence 1)
)
  3. Traceback: starting from the bottom-right cell, the path is followed back to reconstruct the optimal alignment.

The scoring scheme assigns values to matches, mismatches and gaps, which determines how the matrix is filled and which alignment is considered optimal.
