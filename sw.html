Smith-Waterman Algorithm

The Smith-Waterman algorithm finds the optimal alignment of the best matching subsequences between two sequences: it is a local alignment algorithm.

It operates in three steps:

  1. Initialization: a matrix of size (sequence1_length + 1) × (sequence2_length + 1) is created. The first row and column are set to 0.
  2. Fill the matrix: 
    Score[i,j] = max(
    score[i-1, j-1] + s(aᵢ, bⱼ),     # diagonal (match/mismatch)
    score[i-1, j]   + gap_penalty,   # top      (gap in sequence 2)
    score[i, j-1]   + gap_penalty,   # left     (gap in sequence 1)
    0                                # ensures local alignment
)
  3. Traceback: starting from the highest-scoring cell, the path is followed back until a 0 is reached, identifying the best local match.

The scoring scheme assigns values to matches, mismatches and gaps, the same as in Needleman-Wunsch. The key difference is the inclusion of 0 as a minimum, which prevents negative scores and confines the alignment to the best-matching region.
